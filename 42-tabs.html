Installation

Install the CLI tool using npm or your preferred package manager.

npm install -g nexus-cli

Configuration

Initialize your project config file. This creates a nexus.config.js in your root directory.

nexus init --template=ts

Deploy

Push your project to production with a single command. Nexus handles builds, CDN invalidation, and health checks automatically.

nexus deploy --env=production
